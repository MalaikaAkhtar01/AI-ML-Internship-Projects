# AI/ML Internship Projects

## Overview
This repository contains AI/ML internship tasks completed using Python and machine learning techniques.  
The projects focus on data analysis, visualization, predictive modeling, and LLM-based applications using real-world and simulated datasets.

---

## Tasks Included

### Task 1: Iris Dataset Visualization
This task focuses on exploratory data analysis and visualization of the Iris dataset.

**Work Done:**
- Data exploration and understanding dataset structure  
- Scatter plots for feature relationships  
- Histograms for data distribution  
- Box plots for outlier detection  

**Outcome:**
Gained insights into data patterns and relationships between different Iris flower species.

---

### Task 2: Stock Price Prediction
This task involves predicting stock prices using historical market data.

**Work Done:**
- Data collection using Yahoo Finance API (yfinance)  
- Data preprocessing and analysis  
- Linear Regression model implementation  
- Comparison of actual vs predicted values  

**Outcome:**
Built a basic machine learning model for stock trend prediction.

---

### Task 3: Heart Disease Prediction
This task focuses on classification using machine learning to predict heart disease.

**Work Done:**
- Exploratory Data Analysis (EDA)  
- Data preprocessing and feature analysis  
- Logistic Regression model training  
- Model evaluation using Confusion Matrix and ROC Curve  

**Outcome:**
Developed a classification model for predicting heart disease risk.

---

### Task 4: Customer Churn Prediction Pipeline
This task focuses on building an end-to-end machine learning pipeline for predicting customer churn using Scikit-learn.

**Work Done:**
- Data preprocessing using ColumnTransformer  
- Feature scaling and encoding  
- Random Forest Classifier  
- Hyperparameter tuning using GridSearchCV  
- Model export using Joblib  

**Outcome:**
Built a reusable ML pipeline achieving ~80% accuracy for churn prediction.

---

### Task 5: Support Ticket Auto Tagging Using LLM
This task uses a pre-trained language model to automatically classify support tickets into categories.

**Work Done:**
- Implemented zero-shot classification using Hugging Face Transformers  
- Defined multiple ticket categories (Billing, Technical Issue, Refund, etc.)  
- Extracted top 3 probable tags per ticket  
- Compared results using different prompts  

**Outcome:**
Developed an LLM-based text classification system without training a model.

---

### Task 6: Context-Aware RAG Chatbot
This task involves building a Retrieval-Augmented Generation (RAG) chatbot using LangChain and Hugging Face models.

**Work Done:**
- Created knowledge base from custom text data  
- Generated embeddings using SentenceTransformers  
- Stored vectors using FAISS  
- Implemented retrieval-based question answering  
- Integrated FLAN-T5 model for response generation  
- Added conversation memory support  

**Outcome:**
Built an intelligent chatbot capable of answering context-based questions using external knowledge.
